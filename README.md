# 🛡️ SecurePay (Hardened Version)

A secure implementation of the SecurePay application demonstrating defensive programming and secure software development practices.

---

## 📌 Overview

SecurePay Hardened is the improved version of the SecurePay application. The project focuses on eliminating vulnerabilities present in the original version by applying secure coding techniques and modern security controls.

---

## ✨ Features

- Secure Authentication
- Input Validation
- Improved Error Handling
- Secure Session Management
- Payment Processing
- MongoDB Database
